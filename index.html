Stretch CSS

A super simple responsive CSS grid framework. Star Tweet

About

Stretch is a 12 column grid that can have any gutter size you want. Columns can be pushed and pulled to make manoeuvring content really easy and two breakpoint sizes can be specified.

1 Sass file. 3 variables. Lots of win.
