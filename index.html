About

Stretch is a 12 column grid that can have any gutter size you want. Columns can be pushed and pulled to make manoeuvring content really easy. Two breakpoint sizes can be specified.

1 Sass file. 5 variables. Lots of win.

Getting Started

Sass Variables

The best way to use stretch is to include the stretch.scss file in your Sass build. In this file, set the following variables to whatever you want. These are the default values:

  • $gutter: 20px;
  • $wrap: 960px;
  • $row-spacing: 20px;
  • $md-breakpoint: 768px;
  • $sm-breakpoint: 480px;

Build Your Grid

Wrap Your Content

Start by setting a wrap class on the element that will contain all the rows of content. The width for this is set in the $wrap Sass variable.
